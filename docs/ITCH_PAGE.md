## Long description (page body)

Use itch's WYSIWYG; this Markdown maps cleanly.

```markdown
**THE LINE** is a side-scrolling combat-train roguelike. You build the train,
the train fights itself, and you manage power, crew, and abilities in real
time. Enemies spawn from every direction except forward — the train is the
chase fantasy, and you're being chased.

## Core loop

- **Build** — Stack cars (Engine, Weapon, Armor, Crew, Cargo). Mount turrets.
  Stack items on turrets, Binding-of-Isaac style.
- **Run** — Auto-fire turrets handle the moment-to-moment. You handle power
  allocation, crew assignment, slow-time pauses, and active abilities.
- **Return** — Earn Salvage from kills. Spend it at the Hub on permanent
  Tech Tree unlocks: more turret slots, new weapon categories, +10% global
  damage, auto-run mode, idle income, and more.
- **Repeat** — Each run is 15–25 minutes. Death loses the run's items but
  Salvage persists. Auto-run lets the train re-launch on death once the
  Eternal Engine tech is unlocked.

## What's in v1

- **30 turrets** across 5 weapon categories (kinetic, fire, cryo, explosive,
  electric) plus 8 support modules (shields, repair drones, gravity wells,
  Veil siphons).
- **3 turret behavior archetypes:** auto-fire projectiles, continuous beams,
  AOE pulses. Plus passive support auras.
- **Environmental matrix** — fire+forest = wildfire, electric+swamp = chain
  lightning, cryo+snow = doubled slow zones. 5 weapon types × 5 biomes.
- **8-node tech tree** with tier-gated unlocks: Spare Parts, Frostworks,
  Pyrotechnics, Demolitionist, Reinforced Rounds, Storm Caller, Eternal
  Engine, Salvage Magnets.
- **Build sharing** — Copy a URL mid-run that encodes your loadout. Anyone
  opening that link sees an "Import Build" banner at the Hub.
- **4-encounter grammar** — Travel / Swarm / Mini-boss / Boss cycle.
- **Persistent Salvage** — Survives every reload via IndexedDB save (v5
  schema with full v1→v5 migration chain).
- **Idle income** — 1 Salvage per minute while away, capped at 4 hours.
- **Bastion-style slow-time** — Hold SPACE to drop time to 25%.
- **Web Audio SFX** — Procedural oscillator-driven hit, kill, fire, depart,
  and purchase sounds. Mute toggle persisted.
- **First-run tutorial** — Five-card overlay explaining the basics.

## Controls

| Input | Action |
|---|---|
| Mouse | Click Hub buttons; click DEPART to start a run |
| SPACE (hold) | Slow time to 25% |
| Click "Copy Build URL" (in-run) | Share your loadout |
| Click "Abandon Run" (in-run) | Return to Hub |

Turrets auto-target the closest enemy. The player's job is upstream: which
turrets, which items, which power allocation, which crew assignment.

## Roadmap

  stun mechanic, demo-charge re-tuning).
- v2 (Steam) — multi-engine builds, 80-item catalog, item rarity tiers,
  formal synergy transformations, 64-cell environmental matrix, prestige
  system, optional commissioned art pass. Premium $5–7 with 25–40 hours

## Known limitations (v1)

- Engineering Bay panel is a placeholder — turret loadout is fixed at the
  default 8-turret build for canonical layouts. Custom layouts via
  `?b=<token>` import skip the default attach (clean slate, Phase 5.5
  Engineering Bay will own per-loadout module persistence).
- DeathScene exists and respects auto-run, but no in-game event triggers it
  yet (train HP / damage system lands in v1.x).
- Crew Roster + Lore Log + Item Stash are placeholder panels (data layer
  ready, UI in v1.x).
```
